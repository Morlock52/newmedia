Basic Integration Example

This example shows how to integrate the Holographic Media Dashboard into your own application.

Key Features Demonstrated:

  • Initialize the 3D scene
  • Toggle visual effects
  • Control particle systems
  • Monitor performance

Code Example:

// Initialize the dashboard
const dashboard = new HolographicDashboard(container, {
    websocket: {
        url: 'ws://localhost:8080',
        enabled: false // Disable for this example
    },
    particles: {
        count: 500
    },
    performance: {
        adaptiveQuality: true
    }
});

// Initialize and start
dashboard.init().then(() => {
    dashboard.start();
    console.log('Dashboard initialized!');
});
